名称：ChineseFan

技术支持：沐风课堂（http://www.bbvbb.com）

功能介绍：
ChineseFan，是一款用MAXScript脚本语言开发的3dsMax小工具，可以创建扇形2D可编辑样条线拼形阵列。生成的拼图图形可作为挤出图形或放样的截面图形等。
